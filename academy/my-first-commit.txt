git 명령어

------------------

$ git init
- 최초 1회 초기화 (셋업)

$ cd
- 체인지 디렉토리

$ cd ../
- 상위 폴더로

$ cd c:
- c 드라이브로
- tab이랑 같이 사용해서 쉽게 찾을 수 있다

$ git status
- 현재 workgin

$ git add commit 파일명
- Workgin Tree 에서 Staging area 로 이동하게 되고 최종 푸쉬를 기다리는 상태가 된다.

$ git add --all
- 모든 파일을 Staging area 로 이동

$ git push 닉네임 브랜치
- 푸쉬

$ git remote

$ git commit -m "message"
